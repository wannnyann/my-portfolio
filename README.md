# my-portfolio

本リポジトリは、Section6 のポートフォリオ課題をベースに、
フロントエンド・バックエンド・データベースをフルスタックで構築し直したリメイク版です。

## Tech Stack

### Frontend

- Next.js 16.1.10
- TypeScript
- Tailwind CSS
- Node.js

### Backend / Auth

- Firebase Authentication
- Expresss
- TypeScript

### Database

- PostgreSQL

## How to Run(フロント、バックエンドそれぞれ)

npm install  
npm run dev

このプロジェクトは最初にローカルで作成したため、初期段階では main ブランチに直接 push しています。
今回は Docker を使用せず に構築しました。
次回は Docker を用いた環境構築にも挑戦したいと考えています。
